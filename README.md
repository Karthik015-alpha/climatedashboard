# Climate Insights Next.js App

A Next.js App Router experience for climate insights: dashboards, weather/aqi views, predictions, and legacy landing pages.

## Tech Stack
- Next.js 14 (App Router, React, TypeScript)
- Tailwind CSS

## Run Locally
1) `npm install`
2) `npm run dev`

## Build
- `npm run build` then `npm run start`

## Notes
- Legacy FastAPI/Docker/ml_models artifacts were removed; everything now runs inside this Next.js app.
